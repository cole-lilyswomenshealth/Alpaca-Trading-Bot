Webhook Message Templates
BUY — Stock
{"symbol":"TICKER","action":"buy","qty":1,"order_type":"market"}
SELL — Stock
{"symbol":"TICKER","action":"sell","qty":1,"order_type":"market"}
BUY — Crypto
{"symbol":"BTCUSD","action":"buy","qty":1,"order_type":"market"}
SELL — Crypto
{"symbol":"BTCUSD","action":"sell","qty":1,"order_type":"market"}
Replace TICKER with any stock symbol. Crypto uses BTCUSD or ETHUSD. The qty:1 is a placeholder — Fibonacci sizing overrides it automatically.

How It Works
1
TradingView alert fires → sends webhook to server
2
Fibonacci sizing → calculates position size based on sequence
3
Profit protection → sells only when position meets profit threshold
4
Order executes → Alpaca fills the trade, logged to Supabase
Important Notes
• qty:1 in webhooks is a placeholder — Fibonacci sizing overrides it with the correct amount
• Crypto trades 24/7 — BTC and ETH webhooks work on weekends
• Stock market hours — Mon-Fri 9:30 AM - 4:00 PM ET (pre-market 4 AM, after-hours 8 PM)
• Profit protection — set threshold in Inputs tab (currently checks % profit before allowing sell)
